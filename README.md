# Game4AI
Questo spazio verrà usato per caricare gli asset necessari allo sviluppo del gioco. 

**Nota bene:** Ogni gruppo dovrà caricare i propri asset esclusivamente all'interno della relativa cartella assegnata.

---

## Procedura per estrarre il link per Genially
Per importare correttamente i file, la procedura per estrarre il link da seguire **al minimo dettaglio** è:

1. Aprire Github
2. Entrare nel repository
3. Entrare nella cartella del proprio gruppo
4. Scegliere il file che interessa importare su Genially
5. Cliccarlo
6. Fare tasto destro su "raw"
7. Copiare link
8. Incollarlo su Genially
